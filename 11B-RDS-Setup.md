# Step 3 - Create Amazon RDS MySQL

## Objective
Deploy a MySQL database used by the coffee shop application.

## Database Engine
- MySQL

## Configuration Per Tutorial
- Create a new RDS instance
- Configure master username and password
- Place the database in the private database subnets
- Associate the Database Security Group
- Wait until the instance status becomes **Available**

## Verification
- Copy the RDS endpoint.
- This endpoint will be used by the application later.
