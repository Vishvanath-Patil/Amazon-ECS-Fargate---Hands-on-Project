# Step 10 - Create ECS Task Definition

## Objective

Define how the application container should run.

## Configure

- Launch Type: Fargate
- Task Execution Role
- CPU
- Memory
- Container Image (Amazon ECR)
- Port Mapping
- Environment Variables
- AWS Secrets Manager Secret

## Container Definition

Configure:

- Container Name
- Image URI
- Port 80
- Logging
- Environment Variables

## Result

Register the Task Definition.
