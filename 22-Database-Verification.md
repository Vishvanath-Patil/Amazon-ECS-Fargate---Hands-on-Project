# Step 14 - Verify Database Entries

## Objective

Confirm that the application stores submitted data in the MySQL database.

## Steps

1. Open the deployed application.
2. Submit the form.
3. Connect to the Amazon RDS MySQL instance.
4. Open the application database.
5. Query the table used by the application.

## Expected Result

The submitted record should appear in the database.

Verify:
- Name
- Email
- Beverage Choice
