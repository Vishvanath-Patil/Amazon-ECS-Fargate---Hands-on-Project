# Step 8 - Create Amazon ECS Cluster

## Objective

Create an ECS Cluster that will host the application using the **AWS Fargate** launch type.

## Steps

1. Open **Amazon ECS**.
2. Select **Clusters**.
3. Click **Create Cluster**.
4. Choose the **Networking only (AWS Fargate)** option (or Fargate launch type depending on console version).
5. Enter a cluster name.
6. Create the cluster.

## Result

The cluster is ready for deploying ECS Services and Tasks.

---
## Verification

- Cluster Status = Active
- Cluster appears in the ECS console
