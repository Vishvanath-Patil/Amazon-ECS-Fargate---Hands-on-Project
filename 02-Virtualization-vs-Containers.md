# Virtualization vs Containers

## Virtual Machines

- Each VM has its own Guest Operating System.
- Higher isolation.
- Higher resource consumption.
- Slower startup.

## Containers

- Share the host operating system kernel.
- Lightweight.
- Faster startup.
- Portable.
- Better resource utilization.

## Docker Image Flow

Dockerfile
→ Build Image
→ Store Image
→ Run Container
