# Step 1 - Create VPC

The tutorial creates:

- 1 VPC
- 2 Availability Zones
- 2 Public Subnets
- 2 Private App Subnets
- 2 Private Database Subnets
- Internet Gateway
- NAT Gateway
- Route Tables

## Configuration

Availability Zones:
- AZ A
- AZ B

Subnets:
- Public Subnet 1
- Public Subnet 2
- App Subnet 1
- App Subnet 2
- Database Subnet 1
- Database Subnet 2

Associate route tables according to public and private subnet requirements.
